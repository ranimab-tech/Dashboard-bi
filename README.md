# Dashboard-bi
Développement d'un tableau de bord interactif et intuitif, permettant au propriétaire d'un boutique de visualiser facilement l’ensemble de ses ventes, d’analyser les performances des produits et des vendeurs, et de suivre l’évolution du chiffre d’affaires au fil du temps.
Description du projet

Ce projet est une application web permettant de gérer les ventes d’une boutique informatique.
Elle inclut un tableau de bord interactif, une liste dynamique des ventes, des statistiques visuelles, ainsi que plusieurs outils modernes pour faciliter la gestion quotidienne.

L’objectif est d'offrir une interface fluide, intuitive et professionnelle avec :

Visualisation des performances

Gestion simplifiée des ventes

Import/Export de données

Analyse via graphiques

Interface moderne, responsive et rapide

Fonctionnalités principales
Dashboard principal

Affichage des KPI : chiffre d’affaires, ventes totales, meilleur vendeur, produit le plus vendu

Graphiques interactifs (Chart.js) :

Évolution du CA

CA par produit

CA par vendeur

Gestion des ventes

Liste complète des ventes

Recherche instantanée

Pagination

Actions par ligne :

Modifier une vente

Supprimer une vente

Design moderne avec effets hover et boutons ronds colorés

Import / Export

Import de fichiers Excel/CSV (xlsx.js)

Export CSV

Export PDF

Mise à jour automatique des statistiques

Mode clair / sombre

Switch modernisé

Transition douce entre les thèmes

Section IA

Prévision simple du CA

Chat IA simulé pour poser des questions sur les ventes

Feedback utilisateur

Modal animé pour envoyer des suggestions
